# 🏀 微信群篮球活动打卡工具

基于微信云开发的篮球约球打卡小程序，支持**报名/取消/签到/放鸽子标记/积分排行**。

## 📱 功能全景

| 功能 | 说明 |
|------|------|
| 「发起约球」 | 创建活动，设置时间/场地/人数上限 |
| 「报名参加」 | 群友一键报名 |
| 「签到打卡」 | 到场后签到，+10积分 |
| 「取消报名」 | 取消需-5积分 |
| 「放鸽子」 | 发起人可标记未到场者，-20积分 |
| 「积分排行」 | 按积分排序，球王一目了然 |
| 「信誉等级」 | 🏆球王 / ⭐球星 / 👍靠谱 / 🌱新手 / ⚠️警惕 / 🤡鸽子王 |

## 🚀 部署步骤

### 1. 创建微信小程序

在 [微信公众平台](https://mp.weixin.qq.com) 注册小程序，获取 AppID。

### 2. 开通云开发

- 微信开发者工具 → 云开发 → 开通
- 选择环境（建议用正式环境）

### 3. 修改配置

打开 `miniprogram/project.config.json`，将 `appid` 改为你的 AppID。

### 4. 创建云数据库集合

在云开发控制台创建以下集合：

| 集合名 | 权限设置 |
|--------|---------|
| `activities` | 自定义安全规则（见下方） |
| `signups` | 自定义安全规则（见下方） |
| `users` | 所有用户可读，仅自己可写 |

**activities 安全规则：**
```json
{
  "read": true,
  "write": "doc.creatorOpenid === auth.openid || doc.status === 'open'"
}
```

**signups 安全规则：**
```json
{
  "read": true,
  "write": "doc.openid === auth.openid"
}
```

### 5. 上传并部署云函数

在微信开发者工具中：
1. 右键 `cloudfunctions/` → 同步云函数列表
2. 逐个右键每个云函数 → 上传并部署
3. （或用批量部署工具）

### 6. 编译运行

在开发者工具中预览，分享到微信群测试。

## 📊 积分规则

| 行为 | 积分变动 |
|------|---------|
| 报名 + 签到到场 | **+10分** |
| 连续5次到场（额外奖励） | **+15分** |
| 取消报名 | **-5分** |
| 放鸽子（报名未到场） | **-20分** |

### 信誉等级

| 积分 | 称号 | 说明 |
|------|------|------|
| ≥200 | 🏆 球王 | 铁杆球友 |
| ≥100 | ⭐ 球星 | 非常靠谱 |
| ≥50 | 👍 靠谱 | 可以信任 |
| ≥0 | 🌱 新手 | 刚加入 |
| ≥-50 | ⚠️ 警惕 | 需要留意 |
| < -50 | 🤡 鸽子王 | 放鸽子专业户 |

## 🔧 技术栈

- **前端**：WXML + WXSS + JS（原生微信小程序）
- **后端**：微信云函数（Node.js + wx-server-sdk）
- **数据库**：微信云数据库（文档型）
- **云存储**：微信云存储
